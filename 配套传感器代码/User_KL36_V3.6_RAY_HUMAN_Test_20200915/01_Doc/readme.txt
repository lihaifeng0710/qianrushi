原理概述：
任何发热体都会产生红外线，辐射的红外线波长（一般用 μm 描述）跟物体温度有关，表
面温度越高，辐射能量越强。人体因为有恒定的体温，所以会发出特定波长（ 10μm 左右）
的红外线。人体红外传感器通过检测人体释放的红外信号， 能够判断一定范围内是否有人
在活动。传感器默认输出低电平，当检测到人体运动时，会触发高电平输出，小灯亮
（有 3s 左右的延迟）。
