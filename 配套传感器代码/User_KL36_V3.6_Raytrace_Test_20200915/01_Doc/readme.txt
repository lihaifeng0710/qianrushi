原理概述：
当遮挡物体距离传感器红外发射管 2～2.5cm 时，发射管发出的红外射线会被反射回来，
红外接收管打开，模块输出端为高电平，指示灯亮；当红外射线未被反射回来或反射回的强度
不够大时，红外接收管处于关闭状态，模块输出端为低电平，指示灯不亮。
