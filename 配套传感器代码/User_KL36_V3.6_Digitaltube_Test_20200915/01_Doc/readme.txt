原理概述：  
在主函数中通过调用 TM1637_Display（ a,a1,b,b1,c,c1,d,d1）函数可以点亮数码管，其
中数码管的数字显示可在调用函数时设置， a、 b、 c、 d 为要显示的 4 位数字大小；
而 a1、 b1、 c1、d1 为四位数字后面的小数点显示，值为 0 则不显示小数点，值为 1 
则显示小数点。
