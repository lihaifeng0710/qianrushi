原理概述：
加速度传感器首先由前端感应器件感测加速度的大小； 其次由感应电信号器件将其转为可
识别的电信号，这个信号是模拟信号；然后通过 AD 转换器将模拟信号转换为数字信号；
最后通过串口读取数据。
